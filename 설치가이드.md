# 주식 포트폴리오 대시보드 설치 가이드

## ⚠️ 경로 주의사항
모든 배치파일, VBS 파일, 작업 스케줄러가 `C:\Claude\Possession` 경로로 하드코딩되어 있습니다.
**새 PC에도 반드시 동일한 경로에 설치**하세요.

---

## 1단계. Python 설치

1. https://www.python.org/downloads/ 에서 Python **3.10 이상** 다운로드
2. 설치 시 **"Add Python to PATH"** 체크 필수
3. 설치 확인:
   ```
   python --version
   ```

---

## 2단계. 소스코드 다운로드 (GitHub)

```cmd
mkdir C:\Claude
cd C:\Claude
git clone https://github.com/leewanseok/Possession.git
```

> Git이 없으면: https://git-scm.com/download/win 에서 설치 후 진행

---

## 3단계. 가상환경 생성 및 패키지 설치

```cmd
cd C:\Claude\Possession
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

> 설치 완료까지 수 분 소요될 수 있습니다.

---

기존 PC의 `config.json`을 복사하거나, 아래 항목을 직접 입력합니다.

```json
{
  "anthropic_api_key": "sk-ant-...",
  "kis": {
    "app_key": "KIS 앱키",
    "app_secret": "KIS 시크릿",
    "account_no": "계좌번호-01",
    "account_no_us": "계좌번호-01"
  },
  "us_portfolio": {
    "account_no": "계좌번호-01"
  },
  "telegram": {
    "bot_token": "텔레그램 봇 토큰",
    "chat_id": "채팅 ID"
  },
  "portfolio": { ... },
  "coin_portfolio": { ... }

```

> **기존 PC의 config.json을 그대로 복사**하는 것이 가장 간단합니다.

## 설치 확인

1. `run.bat` 실행 → 오류 없이 `Running on http://localhost:5000` 출력되면 정상
2. 브라우저에서 `http://localhost:5000` 접속 → 대시보드 확인
3. 텔레그램 봇에 `/start` 메시지 전송 → 응답 확인

---

## 문제 해결

| 증상 | 해결 방법 |
|------|----------|
| `ModuleNotFoundError` | `.venv\Scripts\activate` 후 `pip install -r requirements.txt` 재실행 |
| KIS API 오류 | `config.json`의 app_key/secret 확인, KIS 개발자센터에서 IP 등록 확인 |
| 포트 5000 사용 중 | `run.bat`이 자동으로 기존 프로세스 종료함 |
| 차트 한글 깨짐 | `C:\Windows\Fonts\malgun.ttf` 존재 여부 확인 (Windows 기본 내장) |
| 작업 스케줄러 등록 실패 | `register_coin_task.bat` 을 관리자로 실행 |
